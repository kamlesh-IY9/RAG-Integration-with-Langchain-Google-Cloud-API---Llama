# RAG with LangChain, Google Cloud API & Llama

A **Retrieval-Augmented Generation** pipeline built with **LangChain**, using **Google Cloud**
services for embeddings/retrieval and a **Llama** model for generation.

## ✨ What it does
- Loads and chunks source documents
- Embeds chunks and retrieves relevant context per query
- Generates grounded answers with a **Llama** LLM via LangChain

## 🛠️ Tech stack
Python · LangChain · Google Cloud API · Llama · vector store

## 🚀 Run it
```bash
git clone https://github.com/kamlesh-IY9/RAG-Integration-with-Langchain-Google-Cloud-API---Llama.git
cd RAG-Integration-with-Langchain-Google-Cloud-API---Llama
pip install -r requirements.txt
# set your API keys / model path here
